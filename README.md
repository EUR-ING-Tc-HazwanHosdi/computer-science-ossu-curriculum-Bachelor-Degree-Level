# Computer Science Curriculum (OSSU-Based)  
### Self-Directed Engineering Program

---

## 🧠 Overview

This repository documents a structured, university-equivalent Computer Science curriculum based on the OSSU (Open Source Society University) framework, supplemented with MIT OpenCourseWare and real-world software engineering projects.

This program is designed to develop **full-stack computer science competency** equivalent to a Bachelor of Computer Science degree through:

- Theoretical computer science foundations  
- Systems-level understanding  
- Algorithmic thinking  
- Real software engineering projects  
- A capstone engineering system (AIMeCHA)

---

## 🎯 Objective

To independently build a complete Computer Science education equivalent to a traditional university degree by combining:

- OSSU structured curriculum  
- MIT / Harvard / Princeton coursework  
- Hands-on engineering projects  
- System design practice  
- Capstone development (AIMeCHA)

---

## 🧱 Curriculum Structure

### 🟢 Foundations
- Programming Fundamentals (CS50 / Python)

- Object-Oriented Programming
- Software Design Principles

### 🟠 Systems
- Computer Architecture
- Operating Systems
- Computer Networking
- Databases

### 🔵 Theory
- Discrete Mathematics
- Algorithm Analysis
- Computability Theory

### 🔴 Applications
- Software Engineering
- System Design
- Distributed Systems (advanced topics)
