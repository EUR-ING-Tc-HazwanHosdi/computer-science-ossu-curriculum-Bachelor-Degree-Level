## 🧪 Reflections & Optimization Logs

### 🔍 Challenge: Handling Input Anomalies
* **Problem:** Early iterations of input scripts would fail or throw errors if the terminal user provided empty strings, special characters, or extreme white spaces.
* **Refactoring Solution:** Implemented robust method chaining (`input().strip().lower()`) right at the data ingestion point. This guarantees predictable string behavior across subsequent comparison blocks without requiring messy nested conditional logic.

### 🔍 Challenge: Custom Script Debugging (`volume_checker.py`)
* **Problem:** Ran into localized syntax and structural execution bugs when validating script pipelines through the terminal.
* **Refactoring Solution:** Isolated logic components into cleanly defined helper functions, ensuring data streams flow predictably from parsing routines directly to output displays.
